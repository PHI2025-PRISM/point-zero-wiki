# 单位概述

SO-08（“猞猁”）为特别行动部直属长期驻防特遣单位，主要负责Site-303-Alpha外围叙事边界稳定、连续性异常遏制、EES污染隔离及高危极地环境下的现实结构维护。

该单位并非传统突击性质特遣队，其核心任务并非正面军事压制，而是：

- 边界异常环境适应
    
- 现实稳定作业
    
- 连续性回收
    
- 狭间渗透处理
    
- 坐标固定
    
- 叙事污染隔离
    

SO-08长期驻防于阿拉斯加州北部Site-303-Alpha冻土层设施群，并承担该设施外围约470公里范围内的一切异常边界响应任务。

由于长期暴露于高浓度EES环境与低稳定叙事区域，该单位成员普遍表现出：

- 情绪平坦化
    
- 时间感迟滞
    
- 社会脱节倾向
    
- 高危环境适应依赖
    
- 现实边界识别强化
    

因此，猞猁成员通常不参与常规社会环境任务。

---

# 战略定位

SO-08被定义为：

# “边界稳定单位（Boundary Stabilization Unit）”

其职责优先级高于：

- 常规收容
    
- 战术压制
    
- 目标清除
    

在AKASHA级以下叙事异常事件中，猞猁拥有局部现实稳定优先授权。

于TARTARUS级以上事件中，猞猁负责：

- 建立锚点网络
    
- 维持连续性通路
    
- 实施区域封冻
    
- 回收高价值存在性目标

# 组织结构

## 1. 指挥组（Command Cell）

负责：

- 叙事风险评估
    
- 边界指数监测
    
- 锚点部署授权
    
- 区域封锁判定
    

指挥权限可临时越级覆盖Site-303-Alpha常规安保协议。

---

## 2. 边界行动组（Boundary Action Teams）

猞猁核心作战力量。

任务包括：

- 狭间渗透
    
- 异常回收
    
- 高危区域搜索
    
- 连续性断裂处理
    

通常以4人小组行动。

标准小组结构：

|职位|职责|
|---|---|
|Lead|边界导航|
|Anchor|锚点部署|
|Observer|现实监测|
|Retrieval|回收执行|

---

## 3. 连续性工程组

负责：

- SCS设备维护
    
- 锚点网络稳定
    
- 局部现实修复
    
- 边界参数校准
    

该组拥有独立现实稳定权限。

---

## 4. 现实锚定组

高危情况下部署。

负责：

- 大规模锚点建立
    
- 区域冻结
    
- 叙事扩散阻断
    

其成员长期暴露于高浓度边界环境，心理损耗率极高。
