# 关于 Point Zero Wiki

## 这个站点是什么？

这是一个使用 [[Obsidian]] + [[Quartz]] 搭建的个人知识库。

## 技术栈

| 工具 | 用途 |
|------|------|
| Obsidian | 本地笔记编辑 |
| Quartz v5 | 静态站点生成 |
| GitHub Pages | 免费托管 |
| GitHub Actions | 自动化部署 |
